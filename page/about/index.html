Who is behind this blog

Hi! I’m Armandas Kaleinykas, a student at Vilnius Tech (VGTU).

This site was built as part of the “Cloud Computing” course, 3rd laboratory work. The whole stack is automated end-to-end:

  • The page is generated by Hugo using the Beautiful Hugo theme.
  • The source lives in a private GitHub repository (hugo).
  • The compiled HTML is pushed to a public repository (dason222.github.io) and served by GitHub Pages for free.

Feel free to browse the blog for cloud notes.
